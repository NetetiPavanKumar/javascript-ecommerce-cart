# 🛒 JavaScript E-Commerce Cart & Checkout (Vanilla JS)

## 📌 Overview

This project is a **frontend e-commerce cart and checkout system** built using **pure (vanilla) JavaScript**, HTML, and CSS.

The primary goal of this project is to **practice and strengthen core JavaScript concepts** such as DOM manipulation, state management, modular code, and browser storage — **without using any frameworks or libraries**.

To improve code organization skills, the same functionality is written in **two different structuring styles**:
- Function-based structure
- Class-based structure

> ⚠️ Note:  
> This project is **not focused on POP vs OOP comparison**.  
> The different approaches are used only to explore **better JavaScript structuring**, while keeping the main focus on **Vanilla JavaScript fundamentals**.

---

## ✨ Features

- Product listing page
- Add to Cart functionality
- Cart quantity update
- Remove items from cart
- Delivery option selection
- Dynamic delivery date calculation
- Order summary calculation:
  - Items total
  - Shipping cost
  - Tax
  - Final order total
- Cart persistence using `localStorage`
- Interactive cart and checkout flow

---

## 🧠 What I Learned

- DOM manipulation using pure JavaScript
- Managing application state using arrays and objects
- Persisting cart data with `localStorage`
- Writing modular JavaScript using ES Modules
- Handling user events and updating UI dynamically
- Structuring JavaScript code for readability and maintainability
- Applying JavaScript concepts to a real-world use case
